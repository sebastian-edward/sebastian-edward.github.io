title: String, StringBuilder and StringBuffer
---

String is immutable in Java. Sometimes to improve performance, some strings need to be mutable, that's the role of StringBuilder and StringBuffer. But when it comes to multithread, StringBuilder can't do great job, only StringBuffer can handle it. StringBuffer use asyn method to allow only one user to change the data, that can avoid race condition.

### String Pool
String Pool is a critical concept in Java. String Pool can make the strings created befored been reused. So if there are many same strings to use, there is no need to redefine them. This can improve the efficiency.

title: Convenient Docker
---

I always have no idea what's the greatness of Docker and why was it blown up to such an extent. Now I can understand some.

It begins when I try to run a large project. There are so many services I need to run before I run the main application, or there will be plenty of errors printed when I run the application. But the configuration and processing process are so annoying and prone to errors. Let alone when you want to run many large applications, you will have mountains of tasks.

But when using Docker, all these pain went away. All I need is a configure file to list the services and corresponding versions. Then it's done, just run this docker container. That saved my world. That's the time when I tasted the convenience of Docker.
